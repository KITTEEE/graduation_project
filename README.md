# graduation_project
毕业设计代码

## 简介
毕业设计做的是一个在线期刊的投稿管理系统，系统分为三个角色：投稿人、审稿人、管理员。功能主要有：
* 投稿人投递稿件以及稿件相关信息的编辑与查看
* 审稿人对稿件进行审核
* 管理员对系统内功能类的增删改查
* 各角色用户的信息管理

**code 目录为前端代码，backend 为后端代码。**

## 系统前端

技术栈： Vue 全家桶 + Ant Design of Vue Pro 组件库 + Axios

### 运行
```
cd code
yarn // 安装依赖
yarn serve // 启动
```

## 系统后端

技术栈： Express + MySQL

### 运行
```
cd backend
npm i // 安装依赖
npm run dev // 启动
```
